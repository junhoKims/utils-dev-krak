# Common

`Common` 문서를 소개합니다 🎉

이 섹션에는 typescript로 작성된 순수 유틸리티 함수로 이루어져 있습니다.

javascript와 typescript 환경 모두 사용 가능합니다.

## 사용

좌측 사이드바에서 함수를 확인하실 수 있습니다.

아래와 같이 모듈을 import 할 수 있습니다.

```js
import { isNil } from 'utils-dev-krak/common';

isNil({});
```
